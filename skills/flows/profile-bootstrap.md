## Profile Bootstrap

1. `.venv/bin/findmejobs profile import --file <resume_path>`
2. `.venv/bin/findmejobs profile show-draft`
3. `.venv/bin/findmejobs profile missing`
4. `.venv/bin/findmejobs profile validate-draft`
5. `.venv/bin/findmejobs profile diff`
6. `.venv/bin/findmejobs profile promote-draft`
7. Optional: `.venv/bin/findmejobs profile set ...` for explicit edits
